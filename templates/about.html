¿Qué es Save The Planet AI?

Es un proyecto educativo que utiliza programación e inteligencia artificial para enseñar sobre el cambio climático de una forma interactiva.

🎯 Objetivo

Crear una herramienta donde las personas puedan aprender sobre el cuidado del planeta mediante juegos, preguntas y consejos.

🌱 Problema

Muchas personas conocen el cambio climático pero no saben cómo sus acciones diarias afectan al ambiente.

💡 Solución

Una aplicación con un juego ambiental y un bot de Discord que responde dudas y comparte información ecológica.

🛠 Tecnologías

  • 🐍 Python
  • 🌐 Flask
  • 🤖 Discord Bot
  • HTML + CSS
