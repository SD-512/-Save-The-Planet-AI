# 🌍 Save The Planet AI

## Este es el tipo de mi proyecto:
> Aplicación web educativa desarrollada con Flask que combina un juego interactivo, un cuestionario sobre el cambio climático, un asistente con inteligencia artificial (EcoGPT) y funciones de reconocimiento y síntesis de voz para fomentar hábitos sostenibles.

---

## Las bibliotecas que voy a utilizar:

- Flask → Crear la aplicación web y las rutas.
- Jinja2 → Generar páginas HTML dinámicas.
- SpeechRecognition → Reconocer la voz del usuario.
- pyttsx3 → Convertir texto en voz.
- python-dotenv → Guardar claves y configuraciones de forma segura.
- SQLite3 → Almacenar preguntas, puntajes y progreso del jugador.
- OpenAI API (si está permitida en el proyecto) → Implementar el asistente EcoGPT.
