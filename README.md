# 🌍 Save The Planet AI

## Esta es la idea de mi proyecto

> Crear una aplicación web interactiva que ayude a concientizar sobre el cambio climático mediante un juego educativo, un cuestionario, inteligencia artificial y herramientas interactivas.

Los usuarios deberán tomar decisiones relacionadas con el cuidado del planeta. Cada decisión tendrá consecuencias positivas o negativas que afectarán el desarrollo del juego.

El proyecto incluirá un sistema de tres vidas, un cuestionario educativo, un asistente inteligente llamado **EcoGPT**, respuestas mediante voz, consejos ambientales y una sección "Antes vs Después" que mostrará visualmente el impacto de las malas decisiones sobre el medio ambiente.

El objetivo es enseñar de forma entretenida cómo pequeñas acciones pueden contribuir a reducir el impacto del cambio climático.

---

## Funciones principales

- 🌍 Juego interactivo.
- ❤️ Sistema de 3 vidas.
- ❓ Cuestionario sobre medio ambiente.
- 🤖 Asistente EcoGPT.
- 🎤 Reconocimiento y síntesis de voz.
- 📷 Comparación "Antes vs Después".
- 🌱 Consejos ecológicos.
- 📊 Estadísticas del jugador.
