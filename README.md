# 🌎 Save The Planet AI

Proyecto desarrollado para un hackathon con el objetivo de concientizar sobre el cambio climático y el cuidado del medio ambiente.

## 🚀 Características

- 🌱 Página web informativa.
- 🎮 Juego educativo.
- 🤖 Bot de Discord interactivo.
- 📚 Información sobre reciclaje, ahorro de agua y energía.
- ❓ Quiz ambiental.

## 🛠️ Tecnologías

- HTML5
- CSS3
- JavaScript
- Python
- Discord.py

## 🤖 Comandos del bot

- !start
- !agua
- !reciclaje
- !energia
- !arboles
- !clima
- !transporte
- !animales
- !eco
- !quiz
- !puntaje

## 📂 Instalación

```bash
git clone https://github.com/TU_USUARIO/TU_REPOSITORIO.git
cd TU_REPOSITORIO
pip install -r requirements.txt
```

Crear un archivo `.env`:

```env
TOKEN=TU_TOKEN_DE_DISCORD
```

Ejecutar:

```bash
python bot.py
```
